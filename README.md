# AfriBuildTrack

AfriBuildTrack is a multi-tenant SaaS construction project management platform designed for construction companies in Africa.

## Core Purpose

Enable construction companies to:

- manage projects
- assign staff
- track project progress
- communicate with clients
- provide real-time visibility

Clients access project dashboards via secure links without creating accounts.

## Platform Roles

- Super Admin (platform owner)
- Company Admin
- Staff
- Client (external, read-only access)

## Architecture (Planned)

Frontend: HTML, CSS, Tailwind, React, Next.js  
Backend: Node.js, Express  
Database: PostgreSQL  
Deployment: Docker, Production Infrastructure
